# Win Rate Factors — Learning Log

## Purpose

This file accumulates patterns from won and lost deals to identify the factors most predictive of deal outcomes. Over time, it enables smarter qualification, more accurate forecasting, and better resource allocation.

## Format

Each entry follows this structure:
```
## [Date] — [Company] — [Won/Lost] — $[Value]
- **MEDDIC at decision:** [Total score /60]
- **Strongest dimension:** [Dimension and score]
- **Weakest dimension:** [Dimension and score]
- **Primary factor:** [What determined the outcome]
- **Pattern match:** [Does this match or break an existing pattern?]
```
